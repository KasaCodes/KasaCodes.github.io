O que o MTA tem a oferecer?

Montador automático para treinos

Com um nome muito intuitivo, o objetivo principal do projeto MTA é a criação de treinos automatizados para academia, facilitando tanto para o aluno quanto para o professor o acesso para o treino montado.

Calculadora IMC

Uma calculadora feita para medir o ÍNDICE DE MASSA CORPORAL, ter este dado facilitará sua visagem de objetivos com a academia. Com o número exato, seu professor ou um nutricionista profissional poderão utilizá-lo para montagem do seu treino e sua dieta.

Integração aluno e professor

O MTA possui integração para alunos e professores, ambos terão acesso ao montador de treinos, com este acesso, ambos podem criar um treino, ou atualizá-lo.
